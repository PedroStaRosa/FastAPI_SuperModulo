# Sistema de registros de chamados.

## Instalação

### 1. Criar a venv

```bash
python -m venv venv
```

### 2. Iniciar a venv

```bash
source venv/Scripts/activate
```

### 3. Instalar as dependências do projeto:

```bash
pip install -r requirements.txt
```

### 4. Ativar a API

```bash
uvicorn main:app --reload
```
